ScaleL
======

モデルのサイズを拡大または縮小するエフェクトです。
既存の Scale エフェクトと比べて、サイズ倍率をわかりやすく指定できることを目指しています。


使い方
-----

まず、エフェクト割当で対象のモデルに ScaleL.fx を割り当ててください。
割り当てるとそのモデルが非表示になってしまいますが、パラメータの設定を終えるとちゃんと表示されるのでご安心ください。
    
次に、ScaleLController.pmx を MMD に読み込んでください。
そして ScaleLController の "スケール倍率" のX座標を調整してください。
この数値がそのままモデルの倍率になります。つまり x=1 なら等倍、x=100 なら100倍になります。

また、モデルのスケールは "スケール冪乗" でも変更することができます。
スケール冪乗の場合、10の冪乗でサイズが大きくなります。
例えば、x=1 ならば10倍、x=2 なら100倍、x=3 なら1000倍になります。

最後に "スケール原点" を設定してください。
基本的には外部親設定を使って、サイズを変更するモデルの "全ての親" と同じ座標に設定しておくのがおすすめです。


モデルごとに倍率を変えたいときは
---------------------------

ScaleLController を使うやり方では、シーン全体で一個の倍率しか指定できません。
サイズを変更したいモデルが複数おり、それぞれに異なる倍率を指定したい場合は
モデル自体に "スケール原点"、"スケール倍率"、"スケール冪乗" ボーンを埋め込んでください。

ScaleL.fx は、割り当てられたモデルに "スケール原点"、"スケール倍率"、"スケール冪乗" ボーンが存在する場合、その値を参照します。

モデルにボーンを追加するには PmxEditor を使ってください。
